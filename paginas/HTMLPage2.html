INVESTIGACION WEB

Red informática.

Es un sistema de dispositivos interconectados (ordenadores, móviles, impresoras) que se comunican para compartir recursos, datos y servicios (archivos, internet, impresoras) mediante conexiones físicas (cables, fibra óptica) o inalámbricas, permitiendo la comunicación eficiente y colaborativa, desde una pequeña LAN hasta la vasta red global de Internet.

Qué es una red LAN.

Una LAN es una red informática que conecta dispositivos en un área geográfica pequeña y limitada, como una casa, oficina o edificio, permitiéndoles compartir recursos (archivos, impresoras) e Internet de forma eficiente, ya sea por cable o inalámbrica, ofreciendo alta velocidad y control de seguridad en un entorno local.

Qué es una red WAN.

Una Red de Área Amplia (WAN), del inglés Wide Area Network, es una red de telecomunicaciones que conecta computadoras y redes locales (LAN) a través de grandes distancias geográficas (ciudades, países), permitiendo la comunicación entre oficinas remotas, centros de datos o usuarios en general, siendo Internet la WAN más grande del mundo; estas redes utilizan tecnologías como fibra óptica, satélites o redes móviles, y son cruciales para la conectividad global empresarial y personal.

Qué es Internet.

Es una vasta red global que conecta millones de computadoras, personas y otros dispositivos en todo el mundo. Nos permite acceder a información desde cualquier lugar, enviar mensajes al instante e interactuar en línea.

Qué es la Web.

La Web (World Wide Web) es un sistema global de documentos y recursos interconectados (páginas web, imágenes, videos) que se accede a través de Internet mediante navegadores, funcionando como una aplicación sobre esta red para compartir información de forma multimedia y enlazada. Es un espacio de contenido (los sitios web) que usa la infraestructura de Internet (la red de computadoras) para funcionar.

Qué es WWW

WWW son las siglas de World Wide Web (Red Mundial de Información en español), un sistema global de documentos interconectados (páginas web) accesibles a través de Internet mediante hipervínculos, que permite navegar por información en línea, a diferencia de Internet, que es la red física subyacente que la hace posible. Fue inventada por Tim Berners-Lee para científicos, pero se convirtió en la forma más popular de usar la red.

Qué es un navegador. Ejemplos.

Un navegador web es una aplicación de software que permite acceder, ver e interactuar con sitios y contenidos de Internet, interpretando el código HTML y otros lenguajes para mostrarlos de forma visual, como texto, imágenes y videos. Los ejemplos más populares incluyen Google Chrome, Microsoft Edge, Mozilla Firefox, Safari y Opera, que funcionan como intermediarios entre el usuario y la web, permitiendo la navegación a través de enlaces y la gestión de favoritos, historial y seguridad.

Qué es un protocolo en informática.

Un protocolo en informática es un conjunto de reglas y estándares que definen cómo los dispositivos se comunican, envían, reciben y procesan datos en una red, asegurando que puedan entenderse entre sí, como un "lenguaje común" para máquinas, especificando formato, sincronización y manejo de errores para una transmisión eficiente y ordenada, como el famoso Protocolo de Internet (IP).

Qué es el protocolo HTTP, HTTPS y DNS

HTTP, HTTPS y DNS son protocolos fundamentales de Internet: HTTP (Protocolo de Transferencia de Hipertexto) permite el intercambio de datos web (texto, imágenes), HTTPS es la versión segura y cifrada de HTTP para proteger la información, y DNS (Sistema de Nombres de Dominio) funciona como un "traductor" que convierte nombres de dominio (ej. google.com) en direcciones IP numéricas para que los navegadores encuentren los sitios, siendo DNS sobre HTTPS (DoH) una forma de cifrar estas consultas DNS para mayor seguridad.

Página web estática vs página web dinámica.

Una página web estática muestra contenido fijo que no cambia a menos que un desarrollador lo modifique manualmente, ideal para información simple y rápida, mientras que una página web dinámica genera contenido en tiempo real, adaptándose a cada usuario y momento mediante bases de datos y lenguajes de programación, permitiendo interactividad como blogs, e-commerce y personalización, aunque puede ser más lenta en carga y costosa de desarrollar.

CMS. Ejemplos. Ventajas e inconvenientes.

Un CMS (Sistema de Gestión de Contenidos) es un software para crear y gestionar sitios web sin programar, como WordPress, Joomla, Drupal, Shopify o Wix, que ofrecen ventajas como facilidad de uso, rapidez de desarrollo, colaboración y SEO, pero presentan desventajas como riesgos de seguridad si no se actualiza, posible lentitud con muchos plugins y costes ocultos.

Proveedor de servicios de Internet. Ejemplos.

Los proveedores de servicios de Internet (ISP) son empresas que te conectan a la red, y ejemplos comunes incluyen grandes operadores como Movistar, Orange, Vodafone, AT&T, Verizon y Comcast (Xfinity), junto con compañías más enfocadas en precios o nichos como Digi, Lowi, Pepephone, O2 y Jazztel, que ofrecen acceso a través de fibra, ADSL o redes móviles para hogares y empresas.

Dirección IP. Diferencia entre una dirección IP privada y pública.

Una dirección IP es un identificador numérico único asignado a cada dispositivo conectado a una red que utiliza el Protocolo de Internet para la comunicación IP pública: Una IP pública es una dirección numérica única y globalmente accesible que identifica tu conexión a Internet, asignada por tu proveedor de servicios de Internet (ISP) a tu router, permitiendo que tus dispositivos se comuniquen con servidores y otros equipos en la red mundial. Funciona como la dirección postal de tu red en Internet, y todos los dispositivos conectados a tu router (computadoras, móviles) comparten esa misma IP pública para salir a la red, aunque tengan IPs privadas dentro de casa. IP privada: Una IP privada es una dirección numérica interna para identificar dispositivos (ordenadores, móviles, etc.) dentro de una red local (LAN), como tu casa u oficina, permitiendo que se comuniquen entre sí sin salir a Internet; estas IPs son asignadas por el router (DHCP), usan rangos reservados (como 192.168.x.x o 10.x.x.x) y son invisibles desde fuera, siendo traducidas a una única IP pública por el router para acceder a Internet, lo que añade seguridad y ahorra direcciones públicas.

¿Qué es el alojamiento web/hosting?¿Que es un dominio? Ejemplos.

El alojamiento web (hosting) es el espacio en un servidor (un ordenador potente siempre conectado a internet) donde se guardan todos los archivos de tu página web (textos, imágenes, videos, código), mientras que el dominio es la dirección única y fácil de recordar (ej: www.google.com) que escribes en el navegador para encontrar esa web, siendo ambos indispensables para que un sitio exista y sea accesible en línea. 🎶.
